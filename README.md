# Rainbow Image Stitching Tool

Quickly stitch multiple PNG images into a rainbow image losslessly, which is used for the `frameAnimatedSpriteType` of **HOI4**.

## UsageAfter 
Creating a work with a timeline in **PHOTOSHOP**, click **File**, **Export**, **Render Video** in sequence. Select `PNG` for **Format**, `5` for **Bit Depth**, and **Straight - No Matte** for **Alpha Channel**. Confirm that the output location of the **Select Folder** is an empty folder, then click **Render**. Open the **Rainbow Image Stitching Tool**, click **Select Folder**, select the folder where you just exported the files, click **Start Stitching and Save**, enter the file name, and the export will be successful.
